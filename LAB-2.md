## **TASK-3**

**Создать вектор с элементами 1, 2, 3, NA, 4, NA, 5, NA**

**Для этого вектора:**

* Вывести все элементы вектора, которые не NA
* Подсчитайте среднее значение всех элементов данного вектора без NA-значений и с NA-значениями

```
v2<-c(1, 2, 3, NA, 4, NA, 5, NA)
v2
```
```
[1]  1  2  3 NA  4 NA  5 NA
```
* Вывести все элементы вектора, которые не NA (not available)
> Функция `na.omit(x)` - исключает отсутствующие значения NA из вектора x:

```
vel.na1<-na.omit(v2)
vel.na1
```
```
[1] 1 2 3 4 5
attr(,"na.action")
[1] 4 6 8
attr(,"class")
[1] "omit"
```

* Вывести сред.значение всех элементов вектора c NA (not available) значениями
> Функция `mean(х)` - находит среднее значение вектора x:
```
vel.na2<-mean(v2)
vel.na2
```
```
[1] NA
```

* Вывести сред.значение всех элементов вектора без NA (not available) значений
> В данном случае функция `mean(х)` содержит аргумент na.rm = TRUE, 
> который позволяет игнорировать отсутствующие значения NA:
```
vel.na3<-mean(v2, na.rm = TRUE)
vel.na3
```
```
[1] 3
```
